# 运行、在源码中打断点

vue 使用的是 rollup 打包。

在 `scripts/config.js` 中的 `genConfig` 方法中的 `config` 对象中加上

```
sourcemap: true
```

然后 `npm run dev`

会打包 `vue.js` 到 `dist`

接下来随便开一个 example ，引入刚刚打包的 `dist/vue.js` ，浏览器直接打开 html ，因为开了 sourcemap ，可以直接在谷歌开发者工具中对源码进行断点
